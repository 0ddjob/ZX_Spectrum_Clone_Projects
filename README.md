# ZX Spectrum Clone Projects
Hardware projects & information for Sinclair ZX Spectrum clone computers

## [Datasheets](/Datasheets)
Relevant datasheets for PCB designs.<br>

## [Belarus Byte DIN to VGA](/Belarus_Byte_DIN-to-VGA)
Convert 5-pin DIN video output to DE15 VGA:

- Pin 1 = synch
- Pin 2 = ground
- Pin 3 = blue
- Pin 4 = green
- Pin 5 = red

## [Leningrad-2 DIN to VGA](/Leningrad_2_DIN-to-VGA)
Convert 5-pin DIN video output to DE15 VGA:

- Pin 1 = audio
- Pin 2 = blue
- Pin 3 = red
- Pin 4 = video
- Pin 5 = green
- Sheath = ground

## [Leningrad 2 Expansion](/Leningrad_2_Expansion)
Experimental board to convert the 2x32 (2.5mm pin pitch) expansion header to standard ZX Spectrum edge connector.

## [Moldova BYTE DIN to VGA](/Moldova_BYTE_DIN-to-VGA)
There are two types of BYTE from Moldova: the "Red BYTE" (in a red case with 7-pin DIN) and the "Speccy BYTE" (looks like a Spectrum with a 5-pin DIN).

- Pin 1 = composite video (not connected)
- Pin 2 = composite synch
- Pin 3 = blue
- Pin 4 = green
- Pin 5 = red
- Pin 6 = ground
- Pin 7 = audio
